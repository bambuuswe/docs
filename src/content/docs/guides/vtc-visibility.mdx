# VTC Visibility

TrucklineMP lets you control who can find and access your Virtual Trucking Company. Choose the visibility level that fits your VTC's purpose.

## Visibility Levels

Every VTC has a visibility setting with three options:

| Visibility | Listed in Directory | Accessible via URL | Who Can View |
|-----------|---------------------|-------------------|--------------|
| **Public** | Yes | Yes | Everyone |
| **Unlisted** | No | Yes | Everyone with the link |
| **Private** | No | No | Members and invited users only |

### Public

Public VTCs are visible to everyone on TrucklineMP.

- Appears in the VTC directory at `trucklinemp.com/vtc`
- Shows up in search results
- Anyone can view the profile
- Anyone can apply if recruitment is open
- Best for: established VTCs looking to grow and recruit publicly

### Unlisted

Unlisted VTCs are hidden from directories but accessible with a direct link.

- Does **not** appear in the VTC directory
- Does **not** show up in search
- Anyone with the URL can view the profile
- Useful for: testing, soft launches, or VTCs that want organic growth through direct links

### Private

Private VTCs are completely hidden and invite-only.

- Does **not** appear in the VTC directory
- Does **not** show up in search
- Profile is only accessible to members
- New members must join through invitation links
- Best for: closed groups, internal teams, or VTCs that want full control over membership

## How to Change Visibility

### During VTC Creation

When creating a new VTC, select the visibility option in the creation form:

1. Go to `trucklinemp.com/vtc/create`
2. Fill in your VTC details
3. Under **Visibility**, choose Public, Unlisted, or Private
4. Complete the creation process

### From VTC Settings

To change visibility after creation:

1. Go to your VTC's management panel
2. Select **General Settings**
3. Scroll to the **Visibility** section
4. Choose Public, Unlisted, or Private
5. Click **Save Configuration**

**Note:** Changing visibility takes effect immediately.

## Invitations for Private VTCs

Private VTCs require invitations for new members to join.

### Creating Invitations

1. Go to your VTC's management panel
2. Select **Invitations** from the sidebar
3. Click **Create Invite**
4. Configure invite options:
   - **Max Uses** — How many times the link can be used (leave empty for unlimited)
   - **Expires In (hours)** — How long the link stays active (leave empty for never)
   - **Target User ID (optional)** — Restrict the invite to a specific user
5. Click **Create Invite**
6. Copy the invite link and share it

### Invite Link Format

```
https://trucklinemp.com/vtc/invite/YOUR_CODE
```

### Managing Invitations

The Invitations page shows all your active and used invites:

| Status | Meaning |
|--------|---------|
| **Active** | Link can still be used |
| **Revoked** | Manually disabled by management |
| **Accepted** | A user joined using this link |
| **Declined** | A user declined the invitation |
| **Expired** | Passed expiration date or max uses reached |

### Revoking Invites

Click **Revoke** next to any active invite to disable it immediately. Revoked links can no longer be used.

### Targeted Invites

Targeted invites are restricted to a specific TrucklineMP user:
- Only that user can accept the invite
- They must be signed in with the correct account
- Useful for recruiting specific drivers

## Visibility vs. Searchable

Visibility controls who can view your VTC. The `searchable` flag controls whether your VTC appears in directory listings and search results.

- **Public + Searchable** — Appears everywhere (default)
- **Public + Not Searchable** — Can be viewed if you have the URL, but not listed
- **Unlisted** — Automatically not searchable
- **Private** — Automatically not searchable

In most cases, you only need to use the visibility setting.

## Who Can View What

| Action | Public | Unlisted | Private |
|--------|--------|----------|---------|
| View profile | Anyone | Anyone with URL | Members only |
| See in directory | Yes | No | No |
| Search | Yes | No | No |
| Apply to join | Yes (if recruitment open) | Yes (if recruitment open) | No |
| Join via invite | No | No | Yes |
| View events | Anyone | Anyone with URL | Members only |
| View news | Anyone | Anyone with URL | Members only |

## Staff Override

TrucklineMP staff with the `web.vtc.private.access` permission can view private VTCs for moderation purposes. This does not make them members of the VTC.

## Best Practices

### Public VTCs
- Keep your profile complete and professional
- Enable recruitment when actively looking for members
- Respond to applications promptly
- Use public visibility once you're ready to grow

### Unlisted VTCs
- Share your URL in targeted communities
- Use for soft launches before going public
- Keep recruitment settings appropriate for your audience

### Private VTCs
- Create targeted invites for specific users
- Set reasonable expiration times for security
- Monitor active invites and revoke unused ones
- Use max uses to control how widely a link spreads

### Switching Visibility

**Public → Private:** Existing members remain members. New members must be invited.

**Private → Public:** Anyone can now find and view your VTC. Make sure your profile is ready.

**Unlisted → Public:** Your VTC will appear in directories and search.

## Troubleshooting

### "My VTC doesn't appear in the directory"
- Check that visibility is set to **Public**
- Verify recruitment is enabled if you expect to appear in recruiting filters
- Wait a few minutes for caches to update

### "People can't join my private VTC"
- Ensure you created an active invitation
- Check that the invite hasn't expired or reached max uses
- For targeted invites, confirm the user is signed in with the correct account

### "My unlisted VTC is showing up in search"
- Unlisted VTCs should not appear in search
- If it does, check that visibility is not set to Public
- Contact support if the issue persists

### "I can't access a private VTC I was invited to"
- Make sure you're signed in
- Check that the invite link hasn't expired
- For targeted invites, use the account the invite was sent to
