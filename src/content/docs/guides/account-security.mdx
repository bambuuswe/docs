# Kontosäkerhet

Hantera din kontosäkerhet, aktiva sessioner och integritetsinställningar för att hålla ditt konto säkert.

## Åtkomst till säkerhetsinställningar

1. Klicka på din avatar i det övre högra hörnet
2. Välj **Inställningar**
3. Välj fliken **Säkerhet**
4. Eller navigera till `trucklinemp.com/user/me/settings?tab=security`

---

## Kopplat konto (Steam)

Ditt Steam-konto är permanent kopplat till ditt TrucklineMP-konto.

- **Status:** Kan inte kopplas bort
- **Syfte:** Primär identitetsverifiering
- **Steam ID:** Visas i det här avsnittet

Denna koppling krävs för kontoverifiering och kan inte ändras.

## Tvåfaktorsautentisering (2FA)

Lägg till ett extra lager av säkerhet till ditt konto.

### Aktivera 2FA

1. Klicka på **Aktivera 2FA**
2. En QR-kod genereras
3. Skanna QR-koden med din autentiseringsapp:
   - Google Authenticator
   - Authy
   - Microsoft Authenticator
   - Valfri TOTP-kompatibel app
4. Eller ange den hemliga nyckeln som visas under QR-koden manuellt
5. Klicka på **Fortsätt till verifiering**
6. Ange den 6-siffriga koden från din app
7. Klicka på **Verifiera & aktivera**
8. **Spara dina reservkoder** — de kommer inte att visas igen

### Reservkoder

När du aktiverar 2FA får du **10 reservkoder**:
- Varje kod kan användas **en gång**
- Använd dem om du tappar åtkomst till din autentiseringsapp
- Förvara dem på en säker plats (lösenordshanterare, utskriven kopia)
- Du kan kopiera alla koder samtidigt

### Inaktivera 2FA

1. Klicka på **Inaktivera 2FA**
2. Ange din autentiseringskod **eller** en reservkod
3. Klicka på **Inaktivera 2FA**
4. 2FA är nu borttagen från ditt konto

**Varning:** Att inaktivera 2FA gör ditt konto mindre säkert.

### Om du tappar bort din autentiseringsapp

1. Använd en reservkod för att logga in
2. När du är inloggad, gå till säkerhetsinställningar
3. Inaktivera 2FA med hjälp av reservkoden
4. Återaktivera 2FA med en ny autentiseringsapp

**Utan reservkoder:** Kontakta supporten via Discord med bevis på identitet.

## Utvecklarläge

Aktivera utvecklarkonsolen för att få tillgång till professionella verktyg.

### Vad du får
- Skapa API-projekt
- Hantera API-uppgifter
- Tillgång till utvecklardokumentation
- Bygg integrationer med TrucklineMP

### Aktivera utvecklarläge
1. Klicka på **Aktivera** i avsnittet för utvecklarläge
2. Läs bekräftelsedialogrutan
3. Klicka på **Bekräfta aktivering**
4. Utvecklarläget är nu aktivt

**Varning:** Denna åtgärd är permanent och går inte att ångra.

### Åtkomst till konsolen
- När det är aktiverat, klicka på **Öppna konsol** för att få tillgång till utvecklarverktyg
- Eller navigera till `trucklinemp.com/developer`

## Aktiva sessioner

Visa och hantera alla enheter som för närvarande är inloggade på ditt konto.

### Sessionsinformation
Varje session visar:
- **Webbläsare** — Chrome, Firefox, Safari, Edge
- **Operativsystem** — Windows, macOS, Linux, Android, iOS
- **IP-adress** — IP-adressen sessionen skapades ifrån
- **Startdatum** — När sessionen skapades
- **Nuvarande märke** — Markerar din nuvarande session

### Återkalla en session
1. Hitta sessionen du vill återkalla
2. Klicka på **Återkalla** bredvid den
3. Sessionen loggas ut omedelbart
4. Enheten måste logga in igen

### Logga ut alla andra sessioner
1. Klicka på **Logga ut alla andra sessioner** högst upp
2. Alla sessioner utom din nuvarande återkallas
3. Användbart om du misstänker obehörig åtkomst

## Bästa praxis för säkerhet

### Grundläggande
- **Aktivera 2FA** — Det enskilt viktigaste säkerhetssteget
- **Spara reservkoder** — Lagra dem i en lösenordshanterare
- **Granska sessioner regelbundet** — Återkalla allt du inte känner igen
- **Använd ett starkt lösenord** — Unikt för TrucklineMP

### Rekommenderat
- Logga ut på delade enheter
- Granska auktoriserade appar periodvis
- Håll dina kopplade konton uppdaterade
- Rapportera misstänkt aktivitet omedelbart

## Felsökning

### "Jag kan inte aktivera 2FA"
- Se till att din autentiseringsapp stöder TOTP
- Kontrollera att din enhets tid är synkroniserad
- Prova en annan autentiseringsapp

### "Jag tappade bort mina reservkoder"
- Om du fortfarande är inloggad, inaktivera 2FA och aktivera det igen
- Om du är utestängd, kontakta supporten via Discord
- Tillhandahåll bevis på identitet för kontoåterställning

### "Jag ser en session jag inte känner igen"
- Återkalla sessionen omedelbart
- Ändra ditt lösenord om möjligt
- Aktivera 2FA om det inte redan är aktiverat
- Kontakta supporten om du misstänker obehörig åtkomst

### "Jag kan inte logga in med 2FA"
- Kontrollera att din autentiseringsapps tid är korrekt
- Prova en reservkod
- Om allt annat misslyckas, kontakta supporten
