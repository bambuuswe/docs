# Kontoradering

Lär dig hur du permanent raderar ditt TrucklineMP-konto och vilken data som tas bort.

## Åtkomst till kontoradering

1. Klicka på din avatar i det övre högra hörnet
2. Välj **Inställningar** (Settings)
3. Välj fliken **Risk** (Danger)
4. Eller navigera till `trucklinemp.com/user/me/settings?tab=danger`

---

## Innan du raderar

### Överväg dessa alternativ
- **Ta en paus** — Du kan helt enkelt sluta använda TrucklineMP utan att radera kontot
- **Exportera din data** — Begär en dataexport innan du raderar kontot
- **Koppla ifrån konton** — Ta bort kopplingar om du planerar att registrera dig på nytt

### Viktiga varningar
- **Denna åtgärd är permanent och oåterkallelig**
- All din data kommer att raderas permanent
- Du kan inte återställa ditt konto efter radering
- Ditt användarnamn och din profil kan bli tillgängliga för andra att använda
- Om tvåfaktorsautentisering är aktiverat måste du ange en verifierings- eller reservkod för att bekräfta raderingen

---

## Vad som raderas

### Permanent borttaget
- Din profil (namn, användarnamn, bild, sociala länkar)
- Alla VTC-medlemskap och roller
- Din aktivitetshistorik
- Foruminlägg och trådar
- Deltagande i evenemang
- Konfigurationsdata
- Personalanteckningar om dig
- Sessionsdata
- Information om kopplade konton

### Vad som kan finnas kvar
- Viss anonymiserad data kan sparas för analysändamål
- Juridiska register kan sparas i enlighet med lagkrav

---

## Hur du raderar ditt konto

### Steg 1: Initiera radering
1. Gå till **Inställningar** > fliken **Risk**
2. Klicka på **Radera konto** (Delete Account)
3. En bekräftelseruta visas

### Steg 2: Vänta på nedräkning
- En 5-sekunders nedräkning startar
- Detta förhindrar oavsiktliga raderingar
- Du kan inte gå vidare förrän nedräkningen är klar

### Steg 3: Bekräfta radering
1. Skriv `DELETE` i bekräftelserutan
2. Raderaknappen aktiveras efter nedräkningen
3. Om du har 2FA aktiverat kommer du att bli ombedd att ange din autentiserings- eller reservkod
4. Klicka på **Radera mitt konto** (Delete My Account)

### Steg 4: Slutförande
- Ditt konto raderas omedelbart
- Du blir utloggad
- Du omdirigeras till startsidan
- En bekräftelsenotis bekräftar raderingen

---

## Efter radering

### Vad händer?
- Du loggas ut från alla sessioner
- Din profilsida kommer att visa ett 404-fel
- Dina VTC-medlemskap tas bort
- Dina foruminlägg och trådar tas bort tillsammans med resten av din data

### Kan jag registrera mig på nytt?
- Ja, du kan skapa ett nytt konto med samma e-postadress
- Din gamla data kommer inte att återställas
- Ditt gamla användarnamn kan vara tillgängligt
- VTC-ägarskap kan inte återställas

---

## Innan radering: Checklista

- [ ] Begär en dataexport
- [ ] Spara all viktig information
- [ ] Anteckna dina reservkoder (om 2FA är aktiverat)
- [ ] Koppla ifrån externa konton (valfritt)
- [ ] Avregistrera dig från e-postutskick

---

## Alternativ till radering

### Om du är frustrerad
- Kontakta support via Discord för att lösa problem
- Ta en paus från plattformen
- Justera dina integritetsinställningar

### Om du vill börja om från början
- Ändra ditt visningsnamn och användarnamn
- Uppdatera din profilbild
- Rensa din aktivitet genom att lämna VTC:er
- Exportera din data först

### Om du är orolig för din integritet
- Dölj externa konfigurationer
- Ändra synlighet för YouTube/Twitch
- Granska och återkalla auktoriserade appar
- Koppla ifrån oanvända anslutningar

---

## Felsökning

### "Jag kan inte radera mitt konto"
- Se till att du har skrivit `DELETE` exakt
- Vänta tills 5-sekundersnedräkningen är klar
- Om du har 2FA aktiverat, kontrollera att du har angett en giltig autentiserings- eller reservkod

### "Jag raderade mitt konto och vill ha tillbaka det"
- Kontoradering är permanent och kan inte ångras
- Du kan skapa ett nytt konto
- Kontakta supporten om du anser att raderingen skedde av misstag
